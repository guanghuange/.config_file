# Mac配置

* 包含zsh、tmux、vim

## 基础配置

    1. 安装brew和brew cask
    
    2. 安装iterm2

        brew cask install iterm2

    3. 安装zsh并设置默认shell为zsh
	
        brew install zsh
        chsh -s /bin/zsh

	重启iterm

    4. 安装oh-my-zsh主题

    5. 安装Powerline fonts
	
	iterm2 设置字体和符号都为Hack
